GISNav downloader

Below you can find websites that provide high resolution orthoimagery in the public domain or with public-domain-like licensing terms.

You should be accessing this page using the onboard browser. The onboard browser will download the files to the onboard /etc/mapserver shared volume by default where the GIS server will be able to pick them up. You can browse to any website to download imagery: as long as you download it using the onboard browser it will be placed in the correct folder. Any GDAL supported raster formats - zipped or unzipped - should work.

  • Earth Explorer - USGS NAIP (USA)
  • MapSite - NLS Orthophotos (Finland)

How did I get here?

You can start the onboard browser and open this page by typing in the following command in your Bash shell:

docker compose -p gisnav up downloader
