GISNav downloader

Below you can find websites that provide high resolution orthoimagery in the public domain or with public-domain-like licensing terms.

You should be accessing this page using the Firefox web browser included with the downloader service. The downloader service will automatically extract the downloaded files to the onboard /etc/mapserver shared volume from where MapServer is able to pick them up. Don't worry - the mapfile will be regenerated when files are added so your MapServer configuration is always up-to-date.

You can browse to any website to download imagery: as long as you download it using the downloader service it will be placed on the shared volume. Any GDAL supported raster formats - zipped or unzipped - should work.

  • Earth Explorer - USGS NAIP (USA)
  • Google Drive - USGS NAIP (USA) - no login
  • MapSite - NLS Orthophotos (Finland)

How did I get here?

You can start the onboard browser and open this page by typing in the following commands in your Bash shell:

cd ~/colcon_ws/src/gisnav/docker
docker compose -p gisnav up downloader
